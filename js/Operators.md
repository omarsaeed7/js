# Operators 
- ** => Power => available in ES8
  - 2**2 = 4
  - 2**3 = 8 

# Logical And Comparison Operators 
```js
|| // OR
&& // AND
< // Smaller Than
> // Bigger Than
<= // Smaller Than or Equal
>= // Bigger Than or Equal
!= // Not Equal ?
== // Is Equal ? in value only
=== // Is Equal in value and data type ?
!z // not true if z is true 
```
# Ternary Operator 
- ? : => 5 > 5 ? 'true' : 'Not true'
